1. в конце работы минифицировать изображения








git remote showorigin  
	проверка на какой удаленный репозиторий установлен наш локальный
git diff
	показывает изменения что мы ввели
git branch nameVetka
	создание новой ветки для работы и т.д.
git branch -d nameVetka
	удаление какой-то (ту имя которой указываем) ветки, если в ветке есть несохраненые данные то гит предупредит
	и откажется удалять, тогда можно вместо маленькой "-d" написать большую "-D" но надо учитывать что оно сотрет
	эту ветку к чертям вместе с всеми несохранеными данными и неважно как сильно мы будем рыдать их уже не вернуть
	назад никак (наверное, я просто не знаю)
git checkout nameVetka
	переход в ту ветку которая нам нужна (из тех что создавали)
git merge nameVetka
	сливаем с веткой мастера(основной) важно знать в какой мы сейчас находимся так как можно слить побочную ветку 
	с веткой мастера, а можно ветку мастера слить с побочной, в какую сливаем та и изменяется
gitk
	спецальная оконная утилита для просмотра развития веток проекта
git log --graph --all --decorate --online
	вывод в консоль графика всех комитов, decorate-цветными делает, online-не выводит лишьнию инфу
